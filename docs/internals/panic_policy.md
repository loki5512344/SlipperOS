# Panic Policy

## Когда паниковать

- Деление на ноль
- Выход за границы массива
- Фатальная ошибка устройства
- User requested panic (команда `panic`)

## Когда НЕ паниковать

- Не найден VirtIO диск → просто не используем
- UART не отвечает → retry
- Не хватило страницы → вернуть None

Panic — всегда красный текст, всегда понятная причина. Никаких silent crash.

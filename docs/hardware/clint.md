# CLINT (Core-Local Interruptor)

- **Адрес**: 0x02000000

## Регистры

| Смещение | Регистр | Описание |
|---|---|---|
| 0x4000 | mtimecmp | Сравнение таймера (hart 0) |
| 0xBFF8 | mtime | Счётчик времени |

## Использование

- Читаем mtime
- Записываем mtimecmp = mtime + delay
- При совпадении — прерывание таймера
- Для перезапуска: пишем новое mtimecmp

Sv39 Virtual Memory

RISC-V 64-bit uses Sv39: 39-bit virtual addresses, 3-level page tables.

Virtual Address (39-bit)

<tr><td>bits 38-30</td><td>bits 29-21</td><td>bits 20-12</td><td>bits 11-0</td></tr>
<table><tr><td>VPN[2] (9)</td><td>VPN[1] (9)</td><td>VPN[0] (9)</td><td>offset (12)</td></tr>
</tbody></table>

Each VPN is an index into a table of 512 entries (8 bytes = 4096 bytes per table).

PTE (Page Table Entry) - 64 bits

BitFieldDescription
0VValid
1RReadable
2WWritable
3XExecutable
4UUser-accessible
5GGlobal
6AAccessed
7DDirty
8RSWReserved for SW
10-53PPN[2:0]Physical Page Number
54-63Reserved

Leaf vs Branch

  • If R/W/X are all 0 → branch (PTE points to next table)
  • If any R/W/X = 1 → leaf (PTE points to physical page)

Any level can be a leaf:

  • L2 leaf → 1GiB page (PPN[2] from PTE, PPN[1]=VPN[1], PPN[0]=VPN[0])
  • L1 leaf → 2MiB page (PPN[2:1] from PTE, PPN[0]=VPN[0])
  • L0 leaf → 4KiB page (PPN[2:0] from PTE)

SATP Register

<tr><td>bits 63-60</td><td>bits 59-44</td><td>bits 43-0</td></tr>
<table><tr><td>MODE (4)</td><td>ASID (16)</td><td>PPN (44)</td></tr>
</tbody></table>
  • MODE = 8 → Sv39
  • PPN = physical address of root table >> 12
  • ASID = Address Space ID (TLB tagging)

map_page

fn map_page(root, vaddr, paddr, flags):
    for level in [2, 1]:
        if PTE is invalid:
            alloc page, set PTE = page_addr | V
        next = PTE.PPN << 12
        advance to next level
    // level 0: set leaf PTE
    PTE = (PPN << 10) | flags | V

# Зачем SlipperOS

Ядро с нуля — не потому что «Linux слишком сложный» (он просто другой), а потому что хочется систему, где всё понятно от boot до shell.

SlipperOS — это:
- Минимум абстракций
- Полный контроль
- Понимание каждого байта

И да, тюлень.

## Почему не Linux

Linux для встраиваемых систем — это 20 миллионов строк и конфиг с тысячей опций. SlipperOS — это пару тысяч строк и один config.toml.

Не для продакшена. Для понимания.
